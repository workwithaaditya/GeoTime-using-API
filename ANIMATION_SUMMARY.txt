## ğŸš€ How to Use

### Default Behavior
Everything works automatically! Just open the app and watch the animations.

### Customize Speed
Edit CSS durations:
```css
animation: slideInLeft 1.2s cubic-bezier(...); /* Change 0.8s to 1.2s */
```

### Customize Easing
Replace easing function:
```css
cubic-bezier(0.34, 1.56, 0.64, 1)  /* Bouncy spring */
cubic-bezier(0.25, 0.46, 0.45, 0.94) /* Smooth */
```

### Disable Animations
Add to JavaScript:
```javascript
document.documentElement.style.setProperty('--animation-duration', '0s');
```
