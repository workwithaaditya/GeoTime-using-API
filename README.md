# SkyWatch - Weather Forecast & Analytics

A comprehensive weather application with real-time data, forecasts, air quality monitoring, and astronomy information.

## Features

### Modern UI
- Glassmorphism design with backdrop blur
- Purple-to-pink gradient backgrounds
- Smooth animations and transitions
- Fully responsive for all devices
- Animated background particles

### Current Weather
- Real-time temperature and conditions
- Feels-like temperature
- Wind speed and direction
- Humidity, visibility, pressure
- UV Index
- Weather condition icons

### 3-Day Forecast
- Daily max/min temperatures
- Weather conditions with icons
- Chance of rain
- Wind speed and humidity
- UV Index

### Hourly Forecast
- Next 12 hours prediction
- Hour-by-hour temperature
- Rain probability
- Wind and humidity conditions

### Astronomy Data
- Sunrise and sunset times
- Moonrise and moonset times
- Moon phases
- Moon illumination percentage

### Air Quality Index
- Real-time air quality monitoring
- AQI level classification
- Detailed pollutant data:
  - Carbon Monoxide (CO)
  - Ozone (O₃)
  - Nitrogen Dioxide (NO₂)
  - Sulphur Dioxide (SO₂)
  - PM2.5 particles
  - PM10 particles

### Weather Alerts
- Government-issued weather warnings
- Severity and urgency levels
- Affected areas
- Alert descriptions

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, gradients, animations)
- JavaScript (ES6+)
- Bootstrap 5.3.8
- Font Awesome 6.4.0
- Google Fonts (Poppins)
- WeatherAPI.com

## API Features

Uses WeatherAPI.com Forecast API with:
- 3-day forecast data
- Air Quality Index (AQI)
- Weather alerts
- Hourly forecasts
- Astronomy data
- Real-time conditions

## Setup

1. Open `index.html` in any modern browser
2. No installation required
3. Requires internet connection for API calls
